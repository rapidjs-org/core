Usage: <start-command> [--dev|-D]

• Commands:

help                            Display help text
start                           ...
stop                            ...
monitor                         Display status of all running proxies and applications

• Flags:

--dev, -D                       Run DEV MODE (PROD MODE otherwise)

• Options:

--wd, -W                        Specify application working directory
--port, -P                      Specify application port
--hostname, -H                  Specify application hostname
--logs, -L                      Specify log file directory

Visit https://rapidjs.org for more information.
