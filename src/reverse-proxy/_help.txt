Usage: <start-command> [--dev|-D]

• Commands:

help                            Display help text
start                           Start a new web space with a specific shell application (successive argument at position 1)
stop                            Stop an existing web space
monitor                         Display status of all running proxies and applications

• Flags:

--dev, -D                       Run DEV MODE (PROD MODE otherwise)

• Options:

--wd, -W                        Specify web space working directory (defaults to process CWD)
--port, -P                      Specify web space port (defaults to 80 (HTTP) / 443 (HTTPS))
--hostname, -H                  Specify web space hostname (defaults to localhost)
--logs, -L                      Specify web space log directory (file logging is disabled by default)

Visit https://rapidjs.org for more information.
