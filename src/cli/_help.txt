Usage: rjs <command> [--<arg:key>|-<arg:shorthand>[ <arg-option>]?]*

→ start                         Start a new host application

  Flags:
  --dev, -D                     Run development mode ('DEV')
  --secure, -S                  Use HTTPS (HTTP by default)

  Options:
  --hostname, -H                Specify hostname (defaults to localhost; multiple hostnames as comma delimited list)
  --port, -P                    Specify listen port (defaults to 80 (HTTP) / 443 (HTTPS))
  --working-dir, -W             Specify application working directory (defaults to process CWD)

→ stop                          Stop an existing host application (or unlink a hostname)

  Options:
  --hostname, -H                Specify associated hostname (defaults to localhost; multiple hostnames as comma delimited list)
  --port, -P                    Specify associated port (defaults to 80 (HTTP) / 443 (HTTPS))

→ monitor                       Monitor running host applications.

→ help                          Display help text

Visit https://rapidjs.org for more information.
